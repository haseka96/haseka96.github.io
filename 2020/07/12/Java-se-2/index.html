Java 视频随笔 Map
Java
Publish Date:   2020-07-12

  1. java.util.Map<k,v>集合

    Map集合的特点：

    • Map集合是一个双列集合，一个元素包含两个值（一个key，一个value）；
    • Map集合中的元素，key和value的数据类型可以相同，也可以不同；
    • Map集合中的元素，key不允许重复，value是可以重复的
    • ap集合中的元素，key和value是一一对应

    java.util.HashMap<k,v>集合 implements Map<k,v>接口

    HashMap集合的特点：

    • HashMap集合底层是哈希表：查询速度快

      • JDK1.8之前：数组+单向链表

      • JDK1.8之后：数组+单向链表/红黑树（链表的长度大于8）：提高了查询的速度

    • hashMap集合是一个无序的集合，存储元素和取出元素的顺序有可能不一致

    java.util.LinedHashMap<k,v>集合 extends HashMap<k,v>集合

    LinkedHashMap的特点：

    • LinkedHashMap集合底层是哈希表+链表（保证迭代的顺序）

    • LinkedHashMap集合是一个有序的集合，存储元素和取出元素的顺序是一致的

  1. Map接口中的常用方法：

    • public V put(K key,V value):把指定的键与指定的值添加到Map集合中。

      • /*
    public V put(K key,V value):把指定的键与指定的值添加到Map集合中。
        返回值：v
            存储键值对的时候，key不重复，返回值v是null
            存储键值对的时候，key重复，会使用新的value替换map中重复的value，返回被替换的value值
 */
private static void show01(){
    //创建Map结合对象，多态
    Map<String,String> map = new HashMap<>();

    String v1 = map.put("霍建华","林心如");
    System.out.println("v1:"+v1);//v1:null

    String v2 = map.put("霍建华","林心如2");
    System.out.println("v2:"+v2);//v2:；林心如

    System.out.println(map);//{霍建华=林心如2}

    map.put("郭靖","黄蓉");
    map.put("杨过","小龙女");
    map.put("梁山伯","祝英台");
    System.out.println(map);//{杨过=小龙女, 霍建华=林心如2, 郭靖=黄蓉, 梁山伯=祝英台}
}

    • public V get(Object key):根据指定的键，在Map集合中获取对应的值。

      • /*
    public V get(Object key):根据指定的键，在Map集合中获取对应的值。
        返回值：
            key存在，返回对应的value值
            key不存在，返回null
 */
private static void  show03(){
    //创建Map集合对象
    Map<String,Integer> map = new HashMap<>();
    map.put("赵丽颖",168);
    map.put("林依晨",165);
    map.put("林志玲",178);
    System.out.println(map);//{林志玲=178, 赵丽颖=168, 林依晨=165}

    Integer v1 = map.get("林依晨");
    System.out.println("v1:"+v1);//v1:165

    Integer v2 = map.get("红莲");
    System.out.println("v2:"+v2);//v2:null
}

    • public Set keyset():获取Map集合中所有的键，存储到Set集合中。

      • Map集合的第一种遍历方式：通过键找值的方式

        Map集合中的方法：public Set keyset():获取Map集合中所有的键，存储到Set集合中。

        实现步骤：

        • 使用Map集合中的方法keySet()，把Map集合所有的键取出来，存储到一个set集合中；
        • 遍历Set集合，获取map集合汇总的每一个key；
        • 通过Map集合中的方法get(key)，通过key找到value

  • public Set<Map.Entry><K,V>> entryset():获取Map集合中所有的键值对对象的集合(Set集合)。

    • Map集合遍历的第二种方法：使用Entry对象遍历

      • Map集合中的方法：public Set<Map.Entry><K,V>> entryset():获取Map集合中所有的键值对对象的集合(Set集合)。
      • 实现步骤：
        • 使用Map集合中的方法entrySet(),把Map集合中的多个Entry对象取出。存储到一个Set集合中；
        • 遍历Set集合，获取每一个Entry对象；
        • 使用Entry对象的方法getKey()和getValue()获取键与值
